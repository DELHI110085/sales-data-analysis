# Sales Data Analysis Project

## Description
This project performs EDA on sales data to extract meaningful business insights. The goal is to understand business patterns and support data-driven decision making.

## Dataset Information
The dataset contains 1000 sales records with the following features:
- product_id
- sale_date
- sales_rep
- sales_amount
- quantity_sold
- product_category
- unit_cost
- unit_price
- customers
- discount
- payment_method
- sales_channel

## Tools Used
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis Performed
- Data cleaning and validation
- Duplicate records check
- Feature engineering (Profit, Month, Year)
- Sales trend analysis
- Region-wise performance analysis
- Product Category analysis
- Sales representative performance
- Payment method analysis

## Key Insights
- Top performing product categories identified based on Sales
- Region-wise sales distribution analyzed
- Sales trends show monthly variation in sales
- Top sales representatives identified based on performance
- Profitability analyzed across different regions
